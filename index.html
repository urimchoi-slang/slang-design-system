Logo & Brand Mark

The Slang wordmark uses Nunito 900 italic with an inline SVG linearGradient fill — never a flat color or raster image. Gradient: pink → violet → blue. When sidebar is collapsed, only the "S" glyph is shown.

Do / Don't
DO

Render via inline SVG with linearGradient fill — ensures true gradient at all sizes on all displays.

The SVG linearGradient is the only correct implementation. It scales perfectly, is accessible, and requires no external assets.

DON'T

Use a flat color, CSS gradient on text, or rasterized PNG/JPG for the wordmark.

Slang

✕ Flat single-color substitution loses the brand gradient entirely.
